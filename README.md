# 🚀 Flask App

A lightweight web application built with Python & Flask.

## 🛠️ Tech Stack
- Python 3.14
- Flask 3.1.3
- Ubuntu (WSL2)

## ⚡ Quick Start

```bash
git clone https://github.com/demarco2016/flask-app.git
cd flask-app
pip install flask
python3 app.py
```
